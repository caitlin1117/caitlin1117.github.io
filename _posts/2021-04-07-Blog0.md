### §1. Load data
Read the Palmer penguins data set. The file has information of 344 penguins, including their species, sex, region, culmen Length, culmen Depth etc. Pandas is a software library written for the Python programming language for data manipulation and analysis. Pandas allows importing data from csv files.


```python
import pandas as pd
url = "https://raw.githubusercontent.com/PhilChodrow/PIC16B/master/datasets/palmer_penguins.csv"
penguins = pd.read_csv(url)
penguins
```

### §2. Basic scatterplot
Matplotlib is a plotting library for Python and Pyplot is a Matplotlib module which provides a MATLAB-like interface.


```python
from matplotlib import pyplot as plt
fig, ax = plt.subplots(1)
```


    
![png](/images/Blog0_files/Blog0_3_0.png)
    


We want to turn the plot above into a scatterplot of the length against the depth of the penguins' culmens. The scatterplot will be a plot of points where each point corresponds to one penguin. We use the `ax.scatter` function. The first argument corresponds to the x-axis and the second corresponds to the y-axis.


```python
ax.set(xlabel = "Culmen Length (mm)",
       ylabel = "Culmen Depth (mm)")
ax.scatter(penguins["Culmen Length (mm)"], penguins["Culmen Depth (mm)"])
fig
```




    
![png](/images/Blog0_files/Blog0_5_0.png)
    



The graph suggests that penguins with longer culmens have shallower culmens, as there are two major clusters of dots, one with deeper but shorter culmens and one with shallower but longer culmens. 

Next we want to further analyze the relationship between culmen length and depth. 

### §3. Scatterplot by species
We will plot each species seperately. In the `ax.scatter` function, we pass a `label` argument to label each species and an `alpha` argument to make the points translucent.


```python
fig, ax = plt.subplots(1)
ax.set(xlabel = "Culmen Length (mm)",
       ylabel = "Culmen Depth (mm)")
species = set(penguins["Species"])
for s in species:
    sPenguins = penguins[penguins["Species"] == s]
    ax.scatter(sPenguins["Culmen Length (mm)"], sPenguins["Culmen Depth (mm)"], alpha=0.5, label=s.split()[0])
ax.legend()
```


    
![png](/images/Blog0_files/Blog0_9_1.png)
    


We notice for each species, there is actually a positive correlation between culmen length and culmen depth, i.e. a penguin with longer culmen tend to have deeper culmen.
